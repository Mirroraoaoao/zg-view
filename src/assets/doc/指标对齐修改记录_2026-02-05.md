## 三、核心改动明细
### 3.1 首页对齐
- 移除 C 位“目标达成进度”图，避免首页出现规划外指标项。
- 重点项目卡片指标名改为 `对接经营调度系统`（首页口径）。
- 业务数据卡片首项改为 `合作区/珠海市重大项目招引` 合并展示。

### 3.2 数据迁移与清理
- 从 `treasuryData` 移除 `fundStatus`。
- 从 `assetData` 移除 `projectSales`、`revitalizedAssets`。
- 在 `operationData` 新增：
  - `fundStatus`（资金整体情况/融资情况/有息负债余额）
  - `assetOperations`（房产销售去化情况（按项目）/本年度已盘活资产规模）
- 为主要模块新增 `hierarchy` 字段，作为层级语义基线。

### 3.3 子屏层级重排
- 财务页：
  - 合并“补充项”到 `L1：核心财务总览`。
  - 保留并明确 `L2：一利五率`、`L2：社会效益类`。
- 资金页：
  - 删除“资金情况”板块，仅保留融资管理相关结构。
- 资产页：
  - 删除“房产销售去化情况（按项目）”“本年度已盘活资产规模”展示。
- 业务数据页：
  - 新增 `L1：资金情况` 与 `L1：资产运营情况`。
  - `项目建设` 更名为 `L1：投资情况`。
  - `人力（业务口径）` 更名为 `L1：人力`。
- 人力页：
  - “人员情况”仅展示三项人数统计。
  - “干部管理”补充 `L2：集团中层干部基本情况`。
  - “职级分布”层级归属调整为“干部管理”下级，不再作为独立 L1。
  - “人员情况-薪酬效能”改为 `L1：人力成本预算执行情况`。

  - 精简为两大主块，移除“重点项目概览”“项目进度说明”。
- 投资页/行政页/经营页：
